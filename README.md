# TPLink IPC Control

A simple custom component to control tplink ipc(set lens mask state for now).

## Usage

1. Add entry in your `configuration.yaml`.
````
tplink_ipc_control:
````

2. Use service `TPLink IPC Control: Set Lens Mask State` with `ipc url`、`username`、`password`.
